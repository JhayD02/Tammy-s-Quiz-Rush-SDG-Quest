                        LEADERBOARD QUICK START GUIDE
                         (5-Minute Overview)
═════════════════════════════════════════════════════════════════════════════


WHAT IS THIS?
─────────────────────────────────────────────────────────────────────────────
A complete local leaderboard system for your Unity quiz game that:
✓ Records player data (Name, School, Date, Score)
✓ Displays the 5 most recent quiz sessions
✓ Shows performance feedback after quiz (Good/Excellent/Needs Improvement)
✓ Saves data locally on the device


SYSTEM FLOW (The journey your player takes)
─────────────────────────────────────────────────────────────────────────────

1. Player enters Quiz Scene
   ↓
2. Player completes quiz
   ↓
3. Results Panel appears ← NEW FEATURE!
   Shows: "Excellent!" (or Good/Needs Improvement)
   Shows: "Final Score: 2450"
   ↓
4. Player returns to Main Menu
   ↓
5. Player clicks "View Leaderboard" button ← NEW FEATURE!
   ↓
6. Leaderboard Scene displays 5 most recent entries
   Shows last player's name, school, date, score
   ↓
7. Player can click "Back to Menu" or "Play Again"


WHAT NEEDS TO BE DONE?
─────────────────────────────────────────────────────────────────────────────

PART 1: Add Results Panel to Quiz Scene (15 min)
   - Create a panel for performance feedback
   - Make it fade in after quiz
   - Show: "Good!", "Excellent!", or "Needs Improvement"

PART 2: Create Leaderboard Scene (30 min)
   - Create new scene called "Leaderboard"
   - Add UI with 5 rows (for 5 recent players)
   - Each row shows: Name | School | Date | Score
   - Add back/play again buttons

PART 3: Add Button to Main Menu (10 min)
   - Add "View Leaderboard" button
   - Make it load Leaderboard scene

COMMON MISTAKES TO AVOID:
─────────────────────────────────────────────────────────────────────────────

❌ Spelling scene names wrong
   ✓ Use: "MainMenu", "Quiz Scene Proper", "Leaderboard"
   ✗ Don't use: "mainmenu", "QuizScene", "leaderboard"

❌ Forgetting to set array sizes to 5
   ✓ Name Texts - set Size: 5
   ✓ School Texts - set Size: 5
   ✓ Date Texts - set Size: 5
   ✓ Score Texts - set Size: 5

❌ Leaving ResultsPanel active
   ✓ Must be INACTIVE (unchecked) in Hierarchy

❌ Not assigning all references
   ✓ Every single one must be assigned
   ✓ Don't leave any empty

❌ Wrong naming convention
   ✓ Name1, Name2, Name3, Name4, Name5
   ✗ Don't use: Name_1, NameText1, etc.


QUICK REFERENCE CARD:
─────────────────────────────────────────────────────────────────────────────

PART 1 - QUIZ SCENE:
  Create: ResultsPanel (Panel)
  Add: CanvasGroup component (Alpha: 0)
  Create: ResultsMessageText (TMP, Size: 48)
  Create: FinalScoreText (TMP, Size: 36)
  Set: ResultsPanel to INACTIVE
  Assign: All references in QuizProper Inspector

PART 2 - LEADERBOARD SCENE:
  Create: New Scene (Leaderboard.unity)
  Create: 5 Record rows (Record1-5)
  Each row has: Name, School, Date, Score (4 texts)
  Create: EmptyStateText
  Create: BackToMenuButton, PlayAgainButton
  Create: LeaderboardManager (attach LocalLeaderboardManager.cs)
  Assign: ALL references in Inspector (5 arrays of size 5 each!)
  Add: Scene to Build Settings

PART 3 - MAIN MENU SCENE:
  Create: ViewLeaderboardButton
  Add: OnClick listener
  Set: Load "Leaderboard" scene
  Done!
