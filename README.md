# Quiz Rush: SDG Quest

An educational, fast-paced quiz game that helps players learn about the 17 Sustainable Development Goals (SDGs) while testing their knowledge in a fun and competitive way.

## About the Game

Quiz Rush: SDG Quest combines the excitement of a timed quiz challenge with educational content about the United Nations' 17 Sustainable Development Goals. Players race against the clock to answer questions correctly, earning points based on accuracy and speed.

## Features

- **Fast-Paced Gameplay**: 15-second timer for each question keeps the game exciting
- **Educational Content**: Learn about all 17 UN Sustainable Development Goals
- **Scoring System**: Earn points for correct answers with time bonuses
- **Interactive Learning**: Dedicated section to study all 17 SDGs before playing
- **Performance Tracking**: View your score, accuracy, and performance feedback
- **Randomized Questions**: Questions are shuffled for a fresh experience each time
- **Responsive Design**: Works on desktop, tablet, and mobile devices

8. Decent Work and Economic Growth
9. Industry, Innovation and Infrastructure
10. Reduced Inequalities
11. Sustainable Cities and Communities
12. Responsible Consumption and Production
13. Climate Action
14. Life Below Water
15. Life on Land
16. Peace, Justice and Strong Institutions
17. Partnerships for the Goals

## How to Play

1. **Open the Game**: Open `index.html` in any modern web browser
2. **Learn or Play**: 
   - Click "Start Quiz" to begin playing immediately
   - Click "Learn About SDGs" to study the goals first
3. **Answer Questions**: Select the correct answer before time runs out
4. **Earn Points**: Get 10 points per correct answer plus time bonuses
5. **View Results**: See your final score, accuracy, and performance rating

## Installation

No installation required! Simply:

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start playing!
