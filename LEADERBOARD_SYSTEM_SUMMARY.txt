WHAT YOU NEED TO DO:
═════════════════════════════════════════════════════════════════════════════

PART 1: SET UP RESULTS PANEL IN QUIZ SCENE (15 minutes)
  1. Create a Panel
  2. Add CanvasGroup component
  3. Add 2 TextMeshPro text fields inside
  4. Set panel to INACTIVE
  5. Assign references to QuizProper script

PART 2: CREATE LEADERBOARD SCENE (30 minutes)
  1. Create new scene: "Leaderboard"
  2. Create 5 record rows (5x4 text fields)
  3. Create EmptyStateText
  4. Create navigation buttons
  5. Create LeaderboardManager with LocalLeaderboardManager script
  6. Assign ALL references in Inspector (critical!)
  7. Add scene to Build Settings

PART 3: ADD BUTTON TO MAIN MENU (10 minutes)
  1. Create button in Main Menu scene
  2. Add OnClick listener to load Leaderboard scene
